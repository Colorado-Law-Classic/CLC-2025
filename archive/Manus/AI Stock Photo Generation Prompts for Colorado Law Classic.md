## Style Guidelines for All Prompts

### Photography Styles to Specify
- Professional sports photography
- Professional event photography
- Professional landscape photography
- Professional product photography
- Professional architectural photography

### Important Elements to Include
- Realistic lighting (specify time of day when relevant)
- No identifiable people (avoid copyright/privacy issues)
- High-quality, professional appearance
- Golf course elements
- Denver/Colorado geographic indicators when appropriate

### Elements to Avoid
- Unrealistic or overly artistic filters
- Specific brand names or logos (except for "Colorado Law Classic")
- Readable text on generated items (can add real text in post-processing)
- Identifiable faces or people
- Unrealistic weather conditions for Colorado in August

## Using These Prompts with AI Image Generators

### Midjourney Specific Tips
- Add "--ar 16:9" for website banner images
- Add "--ar 1:1" for square social media images
- Add "--ar 4:5" for vertical mobile-optimized images
- Add "--v 6" for latest version capabilities
- Add "--style raw" for more photorealistic results

### DALL-E Specific Tips
- Specify "DALL-E 3" for best quality
- Add "professional photography, 8k, ultra detailed" for higher quality
- Request square format for most versatile results

### Stable Diffusion Specific Tips
- Use with LoRA models trained on professional photography
- Add "masterpiece, best quality, highly detailed" to prompts
- Consider using img2img with reference golf course photos

## Post-Processing Recommendations

### Recommended Edits After Generation
1. Add actual "Colorado Law Classic" text and logo
2. Adjust colors to match website color scheme
3. Crop images to required dimensions for different website sections
4. Apply subtle vignetting for more professional look
5. Ensure consistent color grading across all images

### Image Optimization for Web
1. Save as WebP format for best quality/size ratio
2. Compress to under 200KB for fast loading
3. Create multiple sizes for responsive design
4. Add proper alt text when uploading to WordPress
