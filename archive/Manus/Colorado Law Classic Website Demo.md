# Colorado Law Classic Website Demo

This repository contains a complete demo of the Colorado Law Classic charity golf tournament website, built with WordPress and the Kadence theme as requested. This README provides an overview of the demo components and implementation details.

## Demo Components

1. **WordPress Core**: Standard WordPress installation
2. **Kadence Theme**: Modern, lightweight theme with excellent customization options
3. **Custom Layout**: Professional design tailored for charity golf tournaments
4. **Logo & Flyer Integration**: Your provided assets integrated throughout the site
5. **PayPal Integration**: Demo of payment processing for tournament registration
6. **AI-Generated Images**: Custom stock photos for the gallery section
7. **Chat Feature**: Low-cost chat widget implementation

## File Structure

- `index.html` - Main demo page with complete layout and design
- `paypal-integration-demo.php` - PayPal payment processing demo
- `chat-feature-demo.html` - Chat widget implementation demo
- `wp-content/uploads/2025/06/` - Directory containing logo, flyer, and AI-generated images

## Implementation Details

### WordPress & Kadence Theme

The demo uses the Kadence theme for its excellent balance of features:

- Fast performance and lightweight code
- Built-in header and footer builder
- Excellent mobile responsiveness
- Good WooCommerce integration for payment processing
- Active development and regular updates

### Layout & Structure

The website includes the following sections:

- **Header**: Logo and navigation menu
- **Hero Section**: Tournament banner with registration call-to-action
- **Countdown Timer**: Dynamic countdown to tournament date
- **About Section**: Tournament details and information
- **Registration Section**: Player and team registration options
- **Sponsorship Section**: Sponsorship opportunities and pricing
- **Past Sponsors Section**: Recognition of previous sponsors
- **Gallery Section**: Tournament photos (AI-generated for demo)
- **FAQ Section**: Common questions and answers
- **Contact Section**: Contact form and information
- **Footer**: Additional navigation, social links, and newsletter signup

### PayPal Integration

The PayPal integration demo showcases:

- Registration options for individuals ($150) and teams ($600)
- Sponsorship payment options ($1,000-$2,500)
- Secure checkout process
- Order confirmation and receipt
- Transaction management

### AI-Generated Images

The gallery includes five custom-generated images:

1. Golfers teeing off with mountain backdrop
2. Award ceremony with oversized check
3. Aerial view of golf course with sponsor tents
4. Networking scene during lunch break
5. Close-up of branded hole flag with sponsor sign

### Chat Feature

The chat feature demo includes:

- Interactive widget in bottom-right corner
- Pre-programmed responses to common questions
- Documentation for implementing Tidio (recommended) or Tawk.to
- Cost comparison of free vs. premium options
